# Продуктовый магазин
Декстопное приложение, оптимизируеющее работу продуктового магазина и имеющее разделение прав доступа

Всего есть несколько ролей:
- Администратор: имеется возможность создавать новых сотрудников и изменять информацию о старых. Также имеется возможность просмотра истории входа с поиском, а также фильтру по дате.
- Сотрудник: На данный момент имеет возможность авторизироваться, а также изменять информацию о себе в личном кабинете
- Клиент: Имеет возможность просмотра товаров по категориям, выбор товара и добваления его в корзину, оформление заказа с созданием чека в документе word, а также с последующем отправкой на электронную почту

# Стек технологий
- C# / .NET: реализация логики
- WPF: графическое отображение окон, сообщение и прочих элементов
- Material design: стилизация интерфейса
- Entity framework: взаимодействие с базой данных
